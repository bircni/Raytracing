## Usage

Feel free to download the latest release from [releases](https://github.com/bircni/Raytracing/releases) or build it yourself.

### Build

**Requirements**:

- Rust <https://www.rust-lang.org/tools/install>

Build with:

```bash
cargo build --release
```

and run with:

```bash
cargo run --release
```
